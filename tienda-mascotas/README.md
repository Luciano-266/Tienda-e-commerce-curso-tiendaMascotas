🐾 PROYECTO: PetWorld - SPA en React (Luciano Salazar)
-----------------------------------------
Para ejecutar este proyecto en evaluación local:

1. Descomprimir el archivo.
2. Abrir la terminal dentro de la carpeta "tienda-mascotas".
3. Ejecutar el comando: npm install (para restaurar las dependencias).
4. Ejecutar el comando: npm run dev (para levantar el servidor local).

* El proyecto incluye persistencia completa con Local Storage (CRUD funcional en Admin Panel) 
  y consumo de la API externa "catfact.ninja" con manejo estructurado de errores con try/catch.
